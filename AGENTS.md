# Agentinstruktion

Læs både projektets kanoniske Obsidian-vault (`Projekter/_Aktive/Hulensdata/`, start i `00_START_HER.md`) og [`docs/AI-WORKFLOW.md`](docs/AI-WORKFLOW.md) før analyse, dataarbejde eller kodeændringer. Vaulten ejer produkt-, data- og arkitekturbeslutninger; workflow-filen ejer AI-agenternes fælles arbejds- og sikkerhedsprocedure.

Hvis de to reelt er i konflikt, skal konflikten synliggøres og afklares. Ingen af dem må stiltiende overskrive den anden.

Arbejd aldrig direkte på `main`, og godkend eller merge aldrig dit eget arbejde.
