# Claude-instruktion

Læs både projektets kanoniske Obsidian-vault (`Projekter/_Aktive/Hulensdata/`, start i `00_START_HER.md`) og [`docs/AI-WORKFLOW.md`](docs/AI-WORKFLOW.md) først. Vaulten ejer produkt-, data- og arkitekturbeslutninger; workflow-filen er fælles arbejds- og sikkerhedsprocedure for Claude, Codex og andre AI-agenter.

Ved reel konflikt skal forskellen synliggøres og afklares — ikke stiltiende overskrives i nogen retning.

Arbejd på en separat branch, bevar kilder og ukendte værdier, kør Trykpressen og tests før review, og godkend eller merge aldrig dit eget arbejde.
